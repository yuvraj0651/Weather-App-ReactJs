# 🌦️ Weather App 

## 📌 Overview
A real-time Weather Application that provides current weather conditions based on user location or search input using external weather APIs.

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Weather API

---

## ✨ Features
- 🌦️ Real-time weather data
- 🔍 Search by city
- 🌡️ Temperature, humidity, conditions
- 📱 Responsive UI
- 🌙 Dark mode

---

## 🧠 Key Learnings
- API integration
- Handling async data
- UI for real-time data

---
